# 给管理员的全面指南
请在通读并理解玩家规则后再阅读本页。  
本页仅收录管理员需要**额外掌握**的“运作表格与裁断口径”，以便在跑团中快速查阅与执行。

---
## 天气
天气由管理员在**每一个游戏日的傍晚**投掷 `1D12`，并根据结果决定**第二天**的天气。

> 说明：若模组地理环境不支持某类天气（例如沙尘暴），管理员可将该结果替换为更符合地区的极端天气（但应在本次探索中保持一致口径）。
- 1~3 = 晴天
- 4~5 = 多云
- 6~7 = 小雨/小雪
- 8 = 大雨/大雪
- 9 = 雷暴/冰雹
- 10 = 雾气
- 11 = 沙尘暴/暴风
- 12 = 极端天气

## 隔阂值
| 隔阂值  | 效果                      |
| ---- | ----------------------- |
| 0%   | 角色之间不会有交流阻碍             |
| 10%  | 角色之间少数时候会怀疑对方           |
| 20%  | 角色会减少对话的频率              |
| 30%  | 角色之间出现不同选择的时候，意见将难以统一   |
| 40%  | 角色之间互相轻微欺骗对方            |
| 50%  | 角色之间将会回避大量话题            |
| 60%  | 角色之间近乎不交流               |
| 70%  | 角色之间会互相怀疑对方，并且不会信任对方的线索 |
| 80%  | 角色之间有可能发生冲突             |
| 90%  | 角色之间不会互相交换线索            |
| 100% | 角色之间一定会发生冲突             |
